# 「锚点」- 高敏感+ADHD+INFP 专属零内耗成长Web应用

**完全基于咱们所有交流的核心逻辑、方法、避坑红线设计**，100%适配你的特质组合，沿用你最熟悉的「嘉靖-张居正」君臣模型作为UI/UX底层隐喻，无法程序化的部分（比如意义重构、情绪安抚话术），全部以**温柔的、无压力的、贴合你感受的文字提示**做进程序里，做你的专属成长陪伴者。

## 一、应用核心定位与底层隐喻

### 核心定位

- **不是自律工具，是「特质适配的成长陪伴者」**：没有打卡、没有KPI、没有进度条焦虑、没有失败惩罚，只有温柔的提醒、零门槛的功能、可视化的微小胜利。

- **不是任务管理系统，是「灵感-成果-情绪的安全收纳箱」**：帮你接住漫天星光的灵感、固化每一点微小的成长、安抚每一次内耗的情绪，给你一个完全属于自己的、不用讨好任何人的绝对安全区。

### 底层UI/UX隐喻（全程贯穿，降低认知负担）

- **整个应用 = 你的专属「小书房」**：极简、干净、低饱和、无装饰，只有你需要的东西，没有任何干扰。

- **灵感捕手 = 书房门口的「灵感信箱」**：路过就能随手塞一张纸条，不用分类、不用思考，先接住再说。

- **成果收集箱 = 书房里的「成长展示架」**：只放你完成的、值得肯定的东西，没有“未完成”的焦虑。

- **技能台账 = 书房墙上的「技能拼图」**：每掌握一个技能颗粒，就拼上一块拼图，清晰看到自己的成长轨迹，没有“样样松”的自我否定。

- **情绪树洞 = 书房角落的「情绪树洞」**：可以随便说、随便写，不用评判对错，说完就会被温柔接住。

- **睡眠仪式 = 书房里的「睡前小台灯」**：只有在准备睡觉的时候才会打开，给你专属的温柔收尾。

- **应急手册 = 书房抽屉里的「急救小卡片」**：遇到内耗、焦虑、完美主义发作的时候，随手就能抽一张，1分钟就能用。

所有功能都严格遵循「零门槛、无压力、无惩罚、可视化」原则，无法程序化的部分，全部以**温柔的、贴合你感受的文字提示**做进程序里。

### 模块1：首页（灵感信箱入口 + 今日微小胜利展示 + 应急手册入口）

**核心目标**：打开应用直接进入核心功能，无注册登录、无弹窗、无多余跳转，灵感来了立刻就能记录，同时给你今日的微小胜利正反馈，让你一打开就觉得踏实。

#### UI设计（极简干净，低饱和，无装饰）

- 顶部：仅显示应用名「锚点」，右下角是「应急手册」悬浮小图标（像抽屉把手），点击弹出抽屉式应急卡片。

- 中间：90%的区域是**全屏极简灵感输入框**，无标题栏、无分类下拉、无格式工具栏，只有一个占位提示「写下你的灵感/感受/碎碎念...」，光标自动聚焦输入框，打开就能直接打字。

- 底部：仅显示一行**今日微小胜利统计卡片**（比如「今日已记录3条灵感，已完成1个最小闭环」），无复杂图表、无百分比进度，只用数字给你最直观的正反馈，无任何焦虑型设计。

- 折叠区：输入框下方仅保留一个「历史灵感」折叠入口，点击展开按时间倒序展示历史灵感，支持编辑、删除、一键转成果，平时折叠隐藏，不干扰核心输入。

#### 核心功能

1. **10秒极速灵感记录**：输入内容后，点击「保存」/按回车，直接存入数据库，自动记录创建时间，保存成功后清空输入框，弹出**极简弱提示**（比如「灵感已锚定，不会丢啦✨」），不弹窗打断，全程1秒完成。

2. **自动草稿缓存**：输入的内容实时存入Redis，刷新页面、关闭浏览器再打开，草稿依然保留，不会因为走神、误关页面丢失灵感，彻底解决后顾之忧。

3. **一键转成果**：保存灵感后，弹出弱提示「是否转为成果？」，点击即可一键把这条灵感归档到成果收集箱，实现「灵感→落地成果」的闭环。

4. **今日微小胜利自动统计**：自动统计今日记录的灵感数、完成的最小闭环数、掌握的技能颗粒数，只展示正数，不展示“未完成”的内容。

5. **抽屉式应急手册**：点击右下角的抽屉把手，弹出抽屉式应急卡片，卡片上是咱们之前整理的「情绪与内耗应急处理手册」，每个问题对应1个1分钟可执行的解法，解法旁边是**温柔的文字提示**（比如遇到灾难性设想，提示是「这只是你的安全哨兵在过度警惕哦，不是真的会发生，先摸一摸身边的东西，拉回当下吧」）。

### 模块2：灵感信箱（历史灵感分类收纳 + 灵感投喂库入口）

**核心目标**：把你记录的灵感，分类收纳成后续的行动地图，同时给你灵感投喂库，不用等灵感砸脸，主动唤醒创作欲。

#### UI设计

- 顶部：仅显示「灵感信箱」，左侧是「分类标签栏」（5分钟小事库、30分钟兴趣库、长期灵感储备库、情绪树洞库），右侧是「灵感投喂库」入口。

- 主体：极简卡片式布局，按时间倒序展示对应分类的灵感，每张卡片仅显示「时间、内容、标签」，无多余装饰，干净利落。

- 交互：点击卡片可编辑、删除、一键转成果；点击「灵感投喂库」入口，跳转到灵感投喂库页面。

#### 核心功能

1. **自动分类收纳**：记录灵感时，自动弹出**极简弱提示**（比如「要不要给这条灵感加个标签？」），可选标签是咱们之前定的4个，不想加也可以直接跳过，默认归入「长期灵感储备库」。

2. **灵感投喂库**：极简收藏夹式布局，支持一键收藏文章链接、图片、视频、有意思的项目，不用分类太细，能随手存、随手翻就行。闲得慌、没状态、摸鱼的时候，随手翻一翻投喂库，给潜意识提个醒，很多时候翻着翻着，就会突然冒出「我也想做这个」的冲动。

3. **温柔的文字提示**：如果翻了很久投喂库都没灵感，页面底部会弹出**温柔的文字提示**（比如「没关系哦，灵感有时候需要睡一觉才会冒出来，先去做个5分钟的微动作，或者去情绪树洞说说话吧」）。

### 模块3：成长展示架（成果收集箱 + 技能拼图）

**核心目标**：把你所有的成长痕迹，全部可视化、可积累，让你清晰地看到，自己一直在往前走，每一步都算数，彻底告别「一事无成」的自我否定。

#### UI设计

- 顶部：仅显示「成长展示架」，左侧是「成果收集箱」入口，右侧是「技能拼图」入口。

- 成果收集箱页面：极简时间线布局，按时间倒序排列你的所有成果，每条成果仅展示「时间、内容、标签」，无多余装饰，干净利落。右下角保留一个「+ 记录成果」悬浮按钮，点击弹出极简输入框，1步就能记录成果，不用跳转页面。

- 技能拼图页面：极简拼图式布局，把你要学的技能，拆成最细的、可验证的技能颗粒，每个颗粒是一块拼图，未掌握的是灰色，已掌握的是彩色，点击彩色拼图可查看掌握时间，点击灰色拼图可查看「如何掌握这个技能颗粒的文字提示」（比如「@Controller注解的作用」，提示是「可以先把它翻译成你自己的人话，比如「这个类是负责接前端请求的小管家」，然后录成10-30秒的语音，循环听几遍，再写1行测试代码验证，最后记到成果收集箱里哦」）。

#### 核心功能

1. **零门槛成果记录**：弹出的输入框仅需填写成果内容，支持选填1个标签（比如「Java学习」「读书」「项目迭代」），不用写长文、不用排版，点击保存直接加入时间线，零门槛。

2. **标签筛选成果**：时间线顶部仅保留标签筛选栏，点击对应标签，即可筛选该分类下的所有成果，比如只看「Java学习」的成果，清晰看到自己的技术成长轨迹。

3. **技能颗粒度管理**：支持你自己添加、删除、修改技能颗粒，完全由你自己定义，不用符合任何标准，你的感受永远是第一位的。

4. **温柔的文字提示**：如果技能拼图上只有几块彩色，页面底部会弹出**温柔的文字提示**（比如「没关系哦，拼图是一块一块拼起来的，哪怕只有一块，也是你的成长呀，继续加油✨」）。

### 模块5：睡前小台灯（15分钟温柔收尾仪式）

**核心目标**：给你一个专属的温柔收尾，让睡觉从“失败的收尾”，变成“对自己的奖励”，同时给潜意识足够的安全感，不用带着焦虑睡觉。

#### UI设计

- 顶部：仅显示「睡前小台灯」，背景是极浅的暖黄色，营造睡前的温馨氛围，没有任何其他干扰。

- 主体：分4个步骤，每个步骤是一个极简卡片，卡片上是步骤说明和**温柔的文字提示**，完成一个步骤，点击「下一步」，自动进入下一个步骤。

- 交互：完成所有4个步骤后，弹出**极简弱提示**（比如「今天的你已经做得很好了，好好休息吧，明天见✨」），然后自动关闭应用（可选，也可以设置成不关闭）。

#### 核心功能

1. **15分钟温柔收尾仪式**：完全沿用咱们之前整理的仪式，每个步骤都有**温柔的文字提示**：

    - 步骤1：未完成收纳（提示：「把今天所有没做完的事、没落地的灵感、担心的问题，全部写下来吧，不用分类、不用排序，写完就放下，明天再来找它们」）

    - 步骤2：灵感播种（提示：「写下1个你今天最在意、最想深挖的灵感/问题吧，不用展开，只写核心内容，交给你的潜意识，它会在你睡觉的时候帮你找到答案」）

    - 步骤3：最小成就确认（提示：「不管今天多糟糕，必须找出1件你做到的小事哦，哪怕是“今天喝够了水、写了一个灵感关键词、出门走了5分钟”，写下来读一遍，告诉自己“今天的我已经做得很好了”」）

    - 步骤4：明日开门锚定（提示：「只定一件明天要做的、5分钟就能完成的小事吧，不用定全天计划，提前把启动道具摆好，明天醒来，只需要走到书桌前，完成那个已经准备好的动作就行」）

2. **自动计时**：每个步骤自动计时5分钟，时间到了自动进入下一个步骤，也可以手动点击「下一步」，完全由你自己控制。
